    Сайт для фитнес-клуба
    Задача
    Создать сайт для фитнес-клуба. Техническое задание заказчика включало в себя
    1. Главная страница - landing page, задача которой привлекать новых покупателей
    2. Также было необходимо реализовать блоговую систему для публикаций тематических статей
    Для реализации данной задачи были предприняты следующие действия:
    1. Вместе с заказчиком был определен дизайн сайта
    2. Макет был адаптивно сверстан
    3. Для повышения скорости работы было принято решение создать самописную CMS. Функции CMS:
      1. Главная страница проекта - landing page. Для реализации данной задачи необходимо внедрить создание шаблонов для страниц
      2. Так как на всех контентных страницах сайта используется слайдер, то необходимо создать возможность загрузки изображений для слайдера, а так же создание нескольких слайдеров для возможности их использования в любом месте сайта
      3. Блоговая составляющая. Необходимо иметь возможность создавать, изменять и удалять записи
      4. Создание категорий для записей
      5. Поиск записей по заголовку, категории, дате публикации(по конкретной дате, в промежутке времени)
      6. Пользователи могут видеть только записи, разрешенные к просмотру
      7. Поддержка ЧПУ
      8. Разделение пользовательской части и части администратора
      9. Смена контактной информации из панели администратора
      10. Необходимо создать возможность добавлять новых пользователей, обновлять их данные и удалять из базы данные информацию о них
      11. Так как поддержкой сайта может заниматься группа человек, то необходимо реализовать разделение прав пользователей
      12. Для более успешного ведения сайта необходимо получать данные о работе сайта. Для этого создана возможность составления отчетов: статистика просмотра статей за год, статистика добавления статей за год, статистика посещения уникальных пользователей за год
      13. Страница 404 должна отправлять правильные заголовки браузеру
    Сроки
    Установленный срок работы 14 дней
