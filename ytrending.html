    Landing page для продажи услуг продвижения YouTube канала
    Задача
    Это проект для оттачивания своих знаний. Задача проекта заключается в создании Landing Page для продвижения YouTube канала. Для реализации поставленной задачи были выполнены следующие действия:
    1. Создание адаптивной, валидной верстки согласно требованиям сервиса W3C
    2. В целях повышения скорости работы были использованы такие технологии, как:
      1. Верстка по сетке с помощью smart-grid
      2. Использование препроцессора Less
      3. Использование технологии Perfect Pixel для более точного соответствия созданной работы изначальному макету
      4. Использование системы сблорки Gulp с необходимыми пакетами
    3. Использование панели администратора, созданной специально для сайтов типа Landing Page, которая позволяет изменять любые текстовые блоки, а так же блоки с изображениями
