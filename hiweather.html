    Multi-Landing Page для приложения о погоде
    Задача
    Необходимо:
    1. Создать адаптивную верстку согласно макету. Для повышения качества и скорости работы были использованы:
      1. Технология Pixel Perfect, позволяющая создавать верстку наиболее приближенно к заданному макету
      2. Система сборки Gulp
      3. Верстка с помощью сетки smart-grid
      4. Возможности препроцессора Less
    2. Создать тему WordPress, содержащую данную верстку
    3. Добавить возможность создания блоков примеров работы приложения, а так же добавление отзывов из панели администратора
    4. Оптимизировать скорость загрузки, а так же предоставить возможность SEO-специалисту проводить SEO-оптимизацию сайта
