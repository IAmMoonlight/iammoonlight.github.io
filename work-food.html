    Вкусно Жить
    Создать сайт для сохранения рецептов.
    Было необходимо создать сайт для кулинарии. Для заказчика были важны следующие вещи:
    1. Привлекательный дизайн
    2. Возможность создания рецептов
    3. Возможность категоризации рецептов
    4. Возможность поиска рецептов по ингредиентам
    Для реализации установленных требований были предприняты следующие действия:
    1. С заказчиком был утвержден дизайн проекта
    2. Создана верстка проекта. Требования к верстке: адаптивность, валидность согласно сервису W3C
    3. Произведено внедрение данной верстки в Laravel
    4. Произведено разделение сайта на пользовательскую и администраторскую часть
    5. Была реализована блоговая составляющая. Это включает в себя просмотр всех записей, создание, редактирование и удаление записей
    6. Была реализована возможность создания категорий записей для их более удобной сортировки
    7. Была реализована возможность создания меток записей для более удобной фильтрации
    8. Был реализован поиск, включающий возможность поиска по названию записи, категории и одной или нескольким меткам, прикрепленным к записи
