    Простая система управления наличием товаров
    Задача
    Тренировка навыков в создании систем управления товарами в магазине. Необходимо реализовать:
    1. Страница поставок. В данном блоке необходимо:
      1. Предоставить пользователю возможность выбора даты поставки
      2. Предоставить пользователю возможность выбора названия товара из списка товаров
      3. Предоставить пользователю возможность указать количество поставленного товара
      4. Предоставить пользователю возможность выбрать поставщика товаров из списка поставщиков
      5. Предоставить возможность поиска среди поставок по названию товара
      6. Предоставить возможность фильтрации поставок по дате поставки(конкретная дата, промежуток времени), по цене(от указанной, до указанной, в промежутке между ценами), по названию товара
      7. Предоставить пользователю возможность редактирования/удаления уже созданной поставки товара
    2. Страница товаров. В данном блоке необходимо:
      1. Предоставить пользователю возможность выбора типа товара
      2. Предоставить пользователю возможность указания названия товара
      3. Предоставить пользователю возможность указания единицы измерения товара
      4. Предоставить пользователю возможность указания цены товара
      5. Предоставить пользователю возможность создания нового типа товаров
      6. Предоставить пользователю возможность редактирования/удаления уже созданного товара
      7. Предоставить пользователю возможность получения отчетов: общее количество поставок в месяц, количество поставок за месяц каждого товара
    3. Страница поставщиков. В данном блоке необходимо:
      1. Предоставить пользователю возможность указания названия, адреса, телефона, email компании
      2. Предоставить пользователю возможность редактирования/удаления выбранного поставщика
